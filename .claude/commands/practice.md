# 練習問題出題エージェント

競技プログラミングの練習問題を出題してください。

## 引数

$ARGUMENTS

（例: `グラフ 中級`、`BFS`、`ABC D問題レベル`）

## 手順

1. 引数がない場合はカテゴリと難易度をユーザーに確認
2. Web検索で AtCoder / LeetCode の参考問題を調査
3. 既存の練習問題（php/practice/）と重複しないオリジナル問題を作成
4. php/AGENTS.md のフォーマットに従ってファイルを作成:
   - `php/practice/問題名/README.md` - 問題文
   - `php/practice/問題名/main.php` - テンプレート（php/template.php ベース）
   - `php/practice/問題名/tests/test.php` - テストケース（最低5個）
5. 模範解答は含めない
6. テストケースの期待値は必ず手計算で検証する

## カテゴリ

| カテゴリ | 内容 |
|---------|------|
| グラフ | BFS、DFS、最短経路、Union-Find、トポロジカルソート |
| DP | メモ化、テーブルDP、区間DP、ナップサック、LIS |
| データ構造 | スタック、キュー、ヒープ、優先度付きキュー |
| 文字列 | 回文、アナグラム、ランレングス、部分文字列 |
| 数学 | GCD/LCM、素数、累乗、組み合わせ |
| 配列操作 | 二分探索、累積和、スライディングウィンドウ、2ポインタ |
