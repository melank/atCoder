# 括弧の妥当性チェック問題

## 問題

```php
function solution($s) {
}
```

`solution` 関数には、括弧のみで構成された文字列 `$s` が引数として与えられます。

使用される括弧は以下の3種類です：
- 丸括弧: `(` と `)`
- 角括弧: `[` と `]`
- 波括弧: `{` と `}`

文字列 `$s` が **妥当な括弧の組み合わせ** かどうかを判定し、妥当であれば `true`、そうでなければ `false` を返してください。

妥当な括弧の組み合わせとは：
1. すべての開き括弧に対応する閉じ括弧がある
2. 括弧が正しい順序で閉じられている（ネストが正しい）

## 例

### 例1
```
入力 : "()"
出力 : true
```

**解説**: 1組の丸括弧が正しく対応しています。

### 例2
```
入力 : "()[]{}"
出力 : true
```

**解説**: 3種類の括弧がそれぞれ正しく対応しています。

### 例3
```
入力 : "(]"
出力 : false
```

**解説**: 丸括弧の開きに対して角括弧の閉じが来ているため、不正です。

### 例4
```
入力 : "([)]"
出力 : false
```

**解説**: 括弧の順序が正しくありません。`[` を閉じる前に `)` が来ています。

### 例5
```
入力 : "{[]}"
出力 : true
```

**解説**: 波括弧の中に角括弧がネストされており、正しく対応しています。

### 例6
```
入力 : ""
出力 : true
```

**解説**: 空文字列は妥当とみなします。

## 制約

```
0 <= len(s) <= 10000
s は '(', ')', '[', ']', '{', '}' のみで構成される
```

## ヒント

この問題は「スタック（Stack）」というデータ構造を使うと簡単に解けます。

- 開き括弧が来たらスタックに積む（push）
- 閉じ括弧が来たらスタックから取り出し（pop）、対応する開き括弧か確認する
- 最後にスタックが空なら妥当
