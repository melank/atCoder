# 株式売買（Best Time to Buy and Sell Stock）

## 問題

```php
function solution(array $prices): int {
}
```

株価の配列 `$prices` が与えられます。`$prices[i]` は i 日目の株価です。

1回だけ株を購入し、その後売却するとき、得られる**最大利益**を返してください。

利益が出ない場合は `0` を返してください。

## 例

### 例1
```
入力: [7, 1, 5, 3, 6, 4]
出力: 5
```

**解説**: 2日目（価格1）に購入し、5日目（価格6）に売却 → 利益 = 6 - 1 = 5

### 例2
```
入力: [7, 6, 4, 3, 1]
出力: 0
```

**解説**: 価格が下がり続けるため、利益を得られない

### 例3
```
入力: [2, 4, 1]
出力: 2
```

**解説**: 1日目（価格2）に購入し、2日目（価格4）に売却 → 利益 = 2

### 例4
```
入力: [3, 2, 6, 5, 0, 3]
出力: 4
```

**解説**: 2日目（価格2）に購入し、3日目（価格6）に売却 → 利益 = 4

### 例5
```
入力: [1]
出力: 0
```

**解説**: 1日だけでは売買できない

## 制約

```
1 <= count($prices) <= 10000
0 <= $prices[i] <= 10000
```

## ヒント

### 考え方

各日において「今日売るなら、過去の最安値で買っていればよい」

→ **今までの最小値**を追跡しながら走査

### 擬似コード

```
minPrice = ∞
maxProfit = 0

for each price in prices:
    minPrice = min(minPrice, price)
    profit = price - minPrice
    maxProfit = max(maxProfit, profit)

return maxProfit
```

### 計算量

- 時間: O(N)
- 空間: O(1)
