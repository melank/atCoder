# 最小コスト階段問題

## 問題

```php
function solution(array $cost): int {
}
```

各段にコストが設定された階段があります。配列 `cost` が与えられ、`cost[i]` は i 番目の段を踏むときに支払うコストです。

階段の **0番目** または **1番目** からスタートできます。
1回のステップで **1段** または **2段** 登ることができます。

配列の長さを超えた位置が頂上です。**頂上に到達するための最小コスト** を返してください。

## 例

### 例1
```
入力 : cost = [10, 15, 20]
出力 : 15
```

**解説**:
- 1番目（コスト15）からスタート
- 2段登って頂上に到達
- 合計コスト = 15

### 例2
```
入力 : cost = [1, 100, 1, 1, 1, 100, 1, 1, 100, 1]
出力 : 6
```

**解説**:
- 0番目（コスト1）からスタート
- 2段登って2番目（コスト1）
- 2段登って4番目（コスト1）
- 2段登って6番目（コスト1）
- 2段登って8番目... ではなく、1段登って7番目（コスト1）
- 2段登って9番目（コスト1）
- 1段登って頂上
- 合計コスト = 1+1+1+1+1+1 = 6

### 例3
```
入力 : cost = [0, 0, 0, 0]
出力 : 0
```

### 例4
```
入力 : cost = [1, 2]
出力 : 1
```

**解説**: 0番目（コスト1）からスタートして2段登れば頂上。

### 例5
```
入力 : cost = [5]
出力 : 0
```

**解説**: 1番目（存在しない = 頂上）からスタートすればコスト0。

## 制約

```
1 <= count($cost) <= 1000
0 <= cost[i] <= 999
```

## ヒント

### 状態の定義
```
dp[i] = i番目の段に到達するための最小コスト
```

### 漸化式
i番目の段に来るには、(i-1)段目から1段登るか、(i-2)段目から2段登るか:
```
dp[i] = min(dp[i-1], dp[i-2]) + cost[i]
```

### 基底ケース
```
dp[0] = cost[0]  （0番目からスタート）
dp[1] = cost[1]  （1番目からスタート）
```

### 答え
頂上（n番目）に到達するには、(n-1)段目か(n-2)段目から登る:
```
return min(dp[n-1], dp[n-2])
```
