# トリボナッチ数列（Tribonacci Sequence）

## 難易度
★☆☆☆☆（入門）

## アルゴリズム
動的計画法（DP）

## 問題

非負整数 `n` が与えられたとき、`n` 番目のトリボナッチ数を返してください。

トリボナッチ数列は以下のように定義されます：
- T(0) = 0
- T(1) = 1
- T(2) = 1
- T(n) = T(n-1) + T(n-2) + T(n-3)  （n >= 3）

## 入出力例

### 例1
```
入力: n = 0
出力: 0
```

### 例2
```
入力: n = 4
出力: 4
説明: T(4) = T(3) + T(2) + T(1) = 2 + 1 + 1 = 4
      0, 1, 1, 2, 4
```

### 例3
```
入力: n = 5
出力: 7
説明: 0, 1, 1, 2, 4, 7
```

### 例4
```
入力: n = 10
出力: 149
説明: 0, 1, 1, 2, 4, 7, 13, 24, 44, 81, 149
```

## 制約

- 0 <= n <= 37

## ヒント

- フィボナッチと同様のアプローチで解ける
- 3つ前までの値を保持すれば O(1) 空間で解ける
