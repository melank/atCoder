# ソート済み配列の重複削除

## 難易度

初級（★★☆☆☆）

## 問題

昇順にソートされた整数配列 `$nums` が与えられます。

重複する要素を削除し、各要素が1回だけ出現する配列を返す関数 `solution` を実装してください。返す配列も昇順を維持する必要があります。

## 例

### 例1
```
入力: [1, 1, 2]
出力: [1, 2]
```

**解説**: 1 が重複しているので1つに減らします。

### 例2
```
入力: [0, 0, 1, 1, 1, 2, 2, 3, 3, 4]
出力: [0, 1, 2, 3, 4]
```

**解説**: 各要素を1つだけ残します。

### 例3
```
入力: [1, 2, 3]
出力: [1, 2, 3]
```

**解説**: 重複がない場合はそのまま返します。

### 例4
```
入力: [5, 5, 5, 5, 5]
出力: [5]
```

**解説**: 全て同じ値の場合は1つだけ残ります。

### 例5
```
入力: [1]
出力: [1]
```

**解説**: 要素が1つの場合はそのまま返します。

## 制約

```
1 <= count($nums) <= 30000
-10000 <= $nums[i] <= 10000
配列は昇順にソートされている
```

## ヒント

配列が**ソート済み**であることがポイントです。ソートされているため、重複する要素は必ず隣り合っています。

**2ポインタ法**を使うと効率的に解けます：
- 1つ目のポインタ: ユニークな要素を書き込む位置
- 2つ目のポインタ: 配列を走査する位置

時間計算量: O(N)
