# 階段の登り方問題

## 問題

```php
function solution($n) {
}
```

あなたは階段を登ろうとしています。階段は `n` 段あり、頂上に達するまでに登る必要があります。

1回のステップで、**1段** または **2段** 登ることができます。

頂上に達するまでの **異なる登り方の総数** を返してください。

## 例

### 例1
```
入力 : n = 2
出力 : 2
```

**解説**: 2段の階段を登る方法は2通り:
1. 1段 + 1段
2. 2段

### 例2
```
入力 : n = 3
出力 : 3
```

**解説**: 3段の階段を登る方法は3通り:
1. 1段 + 1段 + 1段
2. 1段 + 2段
3. 2段 + 1段

### 例3
```
入力 : n = 4
出力 : 5
```

**解説**: 4段の階段を登る方法は5通り:
1. 1+1+1+1
2. 1+1+2
3. 1+2+1
4. 2+1+1
5. 2+2

### 例4
```
入力 : n = 1
出力 : 1
```

**解説**: 1段なら1通りのみ。

### 例5
```
入力 : n = 5
出力 : 8
```

## 制約

```
1 <= n <= 45
```

## ヒント

この問題は **動的計画法（DP）** の典型問題です。

### 考え方

n段目に到達するには、どこから来たか？
- **(n-1)段目から1段登る**
- **(n-2)段目から2段登る**

つまり:
```
dp[n] = dp[n-1] + dp[n-2]
```

これは **フィボナッチ数列** と同じ構造です！

### 基底ケース
- `dp[1] = 1` （1段なら1通り）
- `dp[2] = 2` （2段なら2通り）

### 注意
再帰で素直に実装すると、同じ計算を何度も繰り返して**指数時間**かかります。**メモ化**または**ボトムアップDP**で効率化しましょう。
