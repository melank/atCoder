# 最大公約数と最小公倍数（GCD / LCM）

## 問題

2つの正の整数が与えられます。最大公約数（GCD）と最小公倍数（LCM）を配列で返してください。

## 関数シグネチャ

```php
function solution(int $a, int $b): array  // [gcd, lcm]
```

## 例

### 例1
```
入力: a = 12, b = 18
出力: [6, 36]
説明:
  - GCD(12, 18) = 6
  - LCM(12, 18) = 36
```

### 例2
```
入力: a = 7, b = 11
出力: [1, 77]
説明: 互いに素なので GCD = 1、LCM = 7 × 11 = 77
```

### 例3
```
入力: a = 100, b = 25
出力: [25, 100]
説明: 25は100の約数
```

### 例4
```
入力: a = 1, b = 1
出力: [1, 1]
```

### 例5
```
入力: a = 48, b = 18
出力: [6, 144]
```

## 制約

- `1 <= a, b <= 10^9`

## ヒント

### ユークリッドの互除法（GCD）
```
GCD(a, b) = GCD(b, a % b)  （b > 0 のとき）
GCD(a, 0) = a
```

### LCM の公式
```
LCM(a, b) = a × b ÷ GCD(a, b)
```

※ オーバーフロー対策として `a ÷ GCD(a, b) × b` の順で計算すると良い
