# 空き時間の検出（Free Time Finder）

## 問題

```php
function solution(array $schedules): array {
}
```

会議の予定 `$schedules` が与えられます（ソートされていない）。

0〜24 の時間帯で、**空いている時間帯**をすべて返してください。

結果は開始時間でソートして返してください。

## 例

### 例1
```
入力: [[9,12], [14,16], [18,20]]
出力: [[0,9], [12,14], [16,18], [20,24]]
```

**解説**:
- 0〜9: 空き
- 9〜12: 会議
- 12〜14: 空き
- 14〜16: 会議
- 16〜18: 空き
- 18〜20: 会議
- 20〜24: 空き

### 例2
```
入力: [[0,5], [3,10], [15,20]]
出力: [[10,15], [20,24]]
```

**解説**: `[0,5]` と `[3,10]` はマージされて `[0,10]`

### 例3
```
入力: [[0,24]]
出力: []
```

**解説**: 終日予定あり、空き時間なし

### 例4
```
入力: []
出力: [[0,24]]
```

**解説**: 予定なし、終日空き

### 例5
```
入力: [[5,10]]
出力: [[0,5], [10,24]]
```

### 例6
```
入力: [[0,8], [18,24]]
出力: [[8,18]]
```

### 例7
```
入力: [[10,12], [8,11], [14,16]]
出力: [[0,8], [12,14], [16,24]]
```

**解説**: `[10,12]` と `[8,11]` がマージされて `[8,12]`

## 制約

```
0 <= count($schedules) <= 1000
0 <= start < end <= 24
時間は整数で与えられる
```

## ヒント

### アルゴリズム

1. **区間マージ**: まず `$schedules` をマージして「使用中」の時間帯を確定
2. **補集合を計算**: マージ結果の「隙間」が空き時間

### 擬似コード

```
// Step 1: 区間マージ
merged = merge($schedules)

// Step 2: 補集合（空き時間）を計算
freeTime = []
prev = 0

for each [start, end] in merged:
    if prev < start:
        freeTime.add([prev, start])
    prev = end

if prev < 24:
    freeTime.add([prev, 24])

return freeTime
```

### 計算量

- 時間: O(N log N) ← マージのソートがボトルネック
- 空間: O(N)
