# 回文判定（Palindrome）

## 問題

文字列が与えられます。英数字のみを考慮し、大文字小文字を区別せずに回文かどうかを判定してください。

回文とは、前から読んでも後ろから読んでも同じ文字列のことです。

## 関数シグネチャ

```php
function solution(string $s): bool
```

## 例

### 例1
```
入力: s = "A man, a plan, a canal: Panama"
出力: true
説明: 英数字のみ抽出 → "amanaplanacanalpanama" → 回文
```

### 例2
```
入力: s = "race a car"
出力: false
説明: 英数字のみ抽出 → "raceacar" → 回文ではない
```

### 例3
```
入力: s = " "
出力: true
説明: 英数字がないので空文字列扱い → 回文とみなす
```

### 例4
```
入力: s = "Was it a car or a cat I saw?"
出力: true
```

### 例5
```
入力: s = "12321"
出力: true
説明: 数字も回文になる
```

## 制約

- `1 <= len(s) <= 200000`
- `s` は ASCII 文字のみで構成される

## ヒント

- 2ポインタ法を使う
- 左端と右端から中央に向かって比較
- 英数字以外はスキップ
- 大文字小文字を統一して比較
