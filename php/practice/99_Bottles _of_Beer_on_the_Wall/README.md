# アナグラム判定問題

## 問題

```php
function solution($s, $t) {
}
```

2つの文字列 `$s` と `$t` が与えられます。

`$t` が `$s` の **アナグラム** であるかどうかを判定し、アナグラムであれば `true`、そうでなければ `false` を返してください。

**アナグラム** とは、文字を並べ替えて作れる文字列のことです。つまり、2つの文字列に含まれる各文字の出現回数がすべて同じであれば、アナグラムです。

## 例

### 例1
```
入力 : s = "anagram", t = "nagaram"
出力 : true
```

**解説**: どちらも a:3, n:1, g:1, r:1, m:1 で同じ構成。

### 例2
```
入力 : s = "rat", t = "car"
出力 : false
```

**解説**: `s` には 't' があるが `t` にはない。

### 例3
```
入力 : s = "listen", t = "silent"
出力 : true
```

**解説**: 同じ文字で構成されている。

### 例4
```
入力 : s = "hello", t = "world"
出力 : false
```

**解説**: 文字の構成が異なる。

### 例5
```
入力 : s = "a", t = "a"
出力 : true
```

**解説**: 同じ1文字。

### 例6
```
入力 : s = "ab", t = "a"
出力 : false
```

**解説**: 長さが異なるのでアナグラムではない。

## 制約

```
1 <= len(s), len(t) <= 10000
s と t は小文字のアルファベット (a-z) のみで構成される
```

## ヒント

### 方法1: ソートして比較
両方の文字列をソートして、同じになればアナグラム。
- 計算量: O(N log N)

### 方法2: 文字の出現回数をカウント
各文字の出現回数をハッシュマップ（連想配列）で数えて比較。
- 計算量: O(N)

PHPでは `count_chars()` や `array_count_values()` が使えます。
