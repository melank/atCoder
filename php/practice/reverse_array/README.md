# 配列の反転

## 難易度

入門（★☆☆☆☆）

## 問題

整数配列 `$nums` を受け取り、配列を**反転**（逆順に並び替え）して返す関数 `solution` を実装してください。

ただし、**新しい配列を作成せず**、元の配列を直接変更して反転させてください（in-place での操作）。

## 例

### 例1
```
入力: [1, 2, 3, 4, 5]
出力: [5, 4, 3, 2, 1]
```

**解説**: 配列の先頭と末尾から順番に要素を交換していきます。

### 例2
```
入力: [1, 2]
出力: [2, 1]
```

**解説**: 要素が2つの場合、先頭と末尾を交換するだけで完了です。

### 例3
```
入力: [7]
出力: [7]
```

**解説**: 要素が1つの場合、反転しても変わりません。

### 例4
```
入力: [1, 2, 3, 4]
出力: [4, 3, 2, 1]
```

**解説**: 要素数が偶数の場合も同様に、先頭と末尾から交換していきます。

## 制約

```
1 <= count($nums) <= 10000
-10000 <= $nums[i] <= 10000
```

## ヒント

配列を反転するには、**2ポインタ法**を使うと効率的です。

1. 左端を指すポインタ `$left = 0`
2. 右端を指すポインタ `$right = count($nums) - 1`
3. `$left < $right` の間、以下を繰り返す:
   - `$nums[$left]` と `$nums[$right]` を交換
   - `$left++`、`$right--`

時間計算量: O(N)、空間計算量: O(1)
