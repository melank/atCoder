# 配列の反転

## 難易度

入門（★☆☆☆☆）

## 問題

整数配列 `$nums` を受け取り、配列を**反転**（逆順に並び替え）して返す関数 `solution` を実装してください。

## 例

### 例1
```
入力: [1, 2, 3, 4, 5]
出力: [5, 4, 3, 2, 1]
```

**解説**: 配列の要素が逆順に並びます。

### 例2
```
入力: [1, 2]
出力: [2, 1]
```

**解説**: 要素が2つの場合、順序が入れ替わります。

### 例3
```
入力: [7]
出力: [7]
```

**解説**: 要素が1つの場合、反転しても変わりません。

### 例4
```
入力: [1, 2, 3, 4]
出力: [4, 3, 2, 1]
```

**解説**: 要素数が偶数の場合も同様に反転されます。

## 制約

```
1 <= count($nums) <= 10000
-10000 <= $nums[i] <= 10000
```

## ヒント

配列を反転するにはいくつかの方法があります：

1. **末尾から取り出す**: 配列の末尾から要素を順番に取り出して新しい配列に追加
2. **2ポインタ法**: 配列の両端から要素を交換しながら中央に向かって進める
3. **組み込み関数**: `array_reverse()` を使用（学習目的では自力実装を推奨）

時間計算量: O(N)
