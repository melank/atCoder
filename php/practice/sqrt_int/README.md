# 平方根の整数部分（Integer Square Root）

## 難易度
★★☆☆☆（初級）

## アルゴリズム
二分探索

## 問題

非負整数 `x` が与えられたとき、`x` の平方根の**整数部分**を返してください。

結果は小数点以下を切り捨てた整数です。

**注意**: `sqrt()` などの組み込み関数は使用しないでください。

## 入出力例

### 例1
```
入力: x = 4
出力: 2
説明: √4 = 2.0
```

### 例2
```
入力: x = 8
出力: 2
説明: √8 = 2.828... → 整数部分は 2
```

### 例3
```
入力: x = 0
出力: 0
```

### 例4
```
入力: x = 1
出力: 1
```

### 例5
```
入力: x = 15
出力: 3
説明: √15 = 3.872... → 整数部分は 3
```

## 制約

- 0 <= x <= 2^31 - 1（約21億）

## ヒント

### なぜ二分探索？
- 「n² ≤ x となる最大の n」を求める問題
- n は 0 から x の範囲で単調増加
- 二分探索で O(log x) で解ける

### アルゴリズム
1. `left = 0`, `right = x` で開始
2. `mid = (left + right) / 2` を計算
3. `mid * mid <= x` なら `left = mid + 1`、答え候補を更新
4. `mid * mid > x` なら `right = mid - 1`
5. `left > right` になったら終了

### 注意点
- `mid * mid` がオーバーフローする可能性がある
- PHP では整数が大きくなると自動で float になるので大丈夫だが、意識しておく
