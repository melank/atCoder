# コイン問題（Coin Change）

## 問題

異なる額面のコインが与えられます。目標金額を作るのに必要な**最小のコイン枚数**を求めてください。

各コインは何枚でも使用できます。目標金額を作ることができない場合は `-1` を返してください。

## 関数シグネチャ

```php
function solution(array $coins, int $amount): int
```

## 例

### 例1
```
入力: coins = [1, 2, 5], amount = 11
出力: 3
説明: 11 = 5 + 5 + 1
```

### 例2
```
入力: coins = [2], amount = 3
出力: -1
説明: 3を作ることはできない
```

### 例3
```
入力: coins = [1], amount = 0
出力: 0
説明: 0円を作るのに必要なコインは0枚
```

### 例4
```
入力: coins = [1, 5, 10, 50, 100, 500], amount = 1234
出力: 11
説明: 1234 = 500×2 + 100×2 + 10×3 + 1×4 = 11枚
```

## 制約

- `1 <= len(coins) <= 12`
- `1 <= coins[i] <= 2^31 - 1`
- `0 <= amount <= 10000`

## ヒント

- 動的計画法（DP）を使います
- `dp[i]` = 金額 `i` を作るのに必要な最小コイン枚数
- 各金額に対して、すべてのコインを試す
