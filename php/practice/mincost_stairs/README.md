# 最小コストで階段を登る（Min Cost Climbing Stairs）

## 難易度
★★☆☆☆（初級）

## アルゴリズム
動的計画法（DP）

## 問題

階段があり、各段にはコスト（体力消費量）が設定されています。
配列 `cost` の `cost[i]` は i 段目に足を踏み入れるときに消費するコストです。

あなたは **1段** または **2段** ずつ登ることができます。
階段の **0段目** または **1段目** からスタートできます。

**階段の一番上（配列の末尾の次）** に到達するまでの最小コストを求めてください。

## 入出力例

### 例1
```
入力: cost = [10, 15, 20]
出力: 15
説明: 1段目（コスト15）からスタートし、2段登って一番上へ
```

### 例2
```
入力: cost = [1, 100, 1, 1, 1, 100, 1, 1, 100, 1]
出力: 6
説明: 0段目→2段目→3段目→4段目→6段目→7段目→9段目→ゴール
      コスト: 1 + 1 + 1 + 1 + 1 + 1 = 6
```

### 例3
```
入力: cost = [0, 0, 0, 1]
出力: 0
説明: 1段目（コスト0）→3段目（コスト0）→ゴール
```

### 例4
```
入力: cost = [5, 10]
出力: 5
説明: 0段目（コスト5）からスタートし、2段登って一番上へ
```

## 制約

- 2 <= len(cost) <= 1000
- 0 <= cost[i] <= 999

## ヒント

### 状態の定義
- `dp[i]` = i段目に到達するための最小コスト

### 遷移式
i段目には「i-1段目から1段」または「i-2段目から2段」で来れる：

```
dp[i] = cost[i] + min(dp[i-1], dp[i-2])
```

### 初期条件
- `dp[0] = cost[0]`（0段目からスタート）
- `dp[1] = cost[1]`（1段目からスタート）

### 答え
- ゴールは n 段目（配列の末尾の次）
- n-1段目 または n-2段目 から1段/2段で到達できる
- `min(dp[n-1], dp[n-2])`
