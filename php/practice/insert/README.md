# 区間の挿入（Insert Interval）

## 問題

```php
function solution(array $intervals, array $newInterval): array {
}
```

**ソート済み**で重複のない区間リスト `$intervals` に、新しい区間 `$newInterval` を挿入してください。

挿入後に重複する区間があればマージし、ソート済みの重複のない区間リストを返してください。

## 例

### 例1
```
入力: intervals = [[1,3], [6,9]], newInterval = [2,5]
出力: [[1,5], [6,9]]
```

**解説**: `[2,5]` は `[1,3]` と重複 → `[1,5]` にマージ

### 例2
```
入力: intervals = [[1,2], [3,5], [6,7], [8,10], [12,16]], newInterval = [4,8]
出力: [[1,2], [3,10], [12,16]]
```

**解説**: `[4,8]` は `[3,5]`, `[6,7]`, `[8,10]` と重複 → `[3,10]` にマージ

### 例3
```
入力: intervals = [], newInterval = [5,7]
出力: [[5,7]]
```

### 例4
```
入力: intervals = [[1,5]], newInterval = [2,3]
出力: [[1,5]]
```

**解説**: `[2,3]` は `[1,5]` に完全に含まれる

### 例5
```
入力: intervals = [[1,5]], newInterval = [6,8]
出力: [[1,5], [6,8]]
```

**解説**: 重複なし、そのまま追加

### 例6
```
入力: intervals = [[3,5], [12,15]], newInterval = [6,6]
出力: [[3,5], [6,6], [12,15]]
```

**解説**: 点としての区間（start == end）

## 制約

```
0 <= count($intervals) <= 10000
$intervals はソート済み（開始点の昇順）
$intervals 内の区間同士は重複していない
0 <= start <= end <= 10000
```

## ヒント

### アプローチ1: 単純な方法

1. `$newInterval` を `$intervals` に追加
2. 通常の区間マージを実行

計算量: O(N log N)

### アプローチ2: 効率的な方法（O(N)）

入力がソート済みなことを活かす:

1. `$newInterval` より**前**の区間をそのまま追加
2. 重複する区間をマージしながら `$newInterval` を更新
3. `$newInterval` より**後**の区間をそのまま追加

```
intervals = [[1,2], [3,5], [6,7], [8,10], [12,16]]
newInterval = [4,8]

Phase 1: [1,2] は [4,8] より前 → 結果に追加
Phase 2: [3,5], [6,7], [8,10] は [4,8] と重複 → [3,10] にマージ
Phase 3: [12,16] は [3,10] より後 → 結果に追加

結果: [[1,2], [3,10], [12,16]]
```

### 計算量

- 時間: O(N)
- 空間: O(N)
