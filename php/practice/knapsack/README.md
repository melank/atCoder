# ナップサック問題

## 問題

```php
function solution(int $capacity, array $weights, array $values): int {
}
```

N個のアイテムがあり、それぞれに**重さ**と**価値**が設定されています。

容量 `capacity` のナップサック（リュック）があります。
ナップサックに入れるアイテムの**重さの合計が容量を超えない**ように選び、**価値の合計を最大化**してください。

各アイテムは**1つずつ**しかなく、**入れるか入れないか**の2択です（0/1ナップサック問題）。

**最大の価値の合計**を返してください。

## 例

### 例1
```
入力 : capacity = 10
       weights = [2, 3, 4, 5]
       values  = [3, 4, 5, 6]
出力 : 13
```

**解説**:
- アイテム0（重さ2, 価値3）を入れる
- アイテム1（重さ3, 価値4）を入れる
- アイテム3（重さ5, 価値6）を入れる
- 合計重さ = 2+3+5 = 10 ≤ 10 ✓
- 合計価値 = 3+4+6 = 13

### 例2
```
入力 : capacity = 7
       weights = [3, 4, 2]
       values  = [4, 5, 3]
出力 : 9
```

**解説**:
- アイテム0（重さ3, 価値4）とアイテム1（重さ4, 価値5）を入れる
- 合計重さ = 3+4 = 7 ≤ 7 ✓
- 合計価値 = 4+5 = 9

### 例3
```
入力 : capacity = 5
       weights = [6, 7, 8]
       values  = [10, 20, 30]
出力 : 0
```

**解説**: どのアイテムも重すぎて入らない。

### 例4
```
入力 : capacity = 50
       weights = [10, 20, 30]
       values  = [60, 100, 120]
出力 : 220
```

**解説**:
- アイテム1（重さ20, 価値100）とアイテム2（重さ30, 価値120）を入れる
- 合計重さ = 20+30 = 50 ≤ 50 ✓
- 合計価値 = 100+120 = 220

### 例5
```
入力 : capacity = 10
       weights = [5]
       values  = [10]
出力 : 10
```

**解説**: 1つだけ入れる。

## 制約

```
1 <= capacity <= 100
1 <= count($weights) <= 20
weights[i] >= 1
values[i] >= 1
count($weights) === count($values)
```

## ヒント

### 状態の定義
```
dp[i][w] = 最初のi個のアイテムを考慮し、
           容量wのナップサックで得られる最大価値
```

### 漸化式
i番目のアイテム（0-indexed）を入れるか入れないか:

```
dp[i+1][w] = max(
    dp[i][w],                              // i番目を入れない
    dp[i][w - weights[i]] + values[i]      // i番目を入れる（入れられる場合）
)
```

### 基底ケース
```
dp[0][w] = 0  （アイテムが0個なら価値は0）
```

### 答え
```
return dp[n][capacity]
```

### 計算量
- 時間: O(N × capacity)
- 空間: O(N × capacity)、最適化すれば O(capacity)
