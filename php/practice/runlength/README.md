# ランレングス圧縮

**難易度**: ★★☆☆☆（初級）

**カテゴリ**: 文字列（ランレングスエンコード）

## 問題

ランレングス圧縮（Run-Length Encoding）は、連続する同じ文字を「文字+出現回数」の形式で表現する圧縮方法です。

与えられた文字列をランレングス圧縮した結果を出力してください。

### ルール

- 連続する同じ文字は「文字 + 連続回数」の形式で表す
- 連続回数が 1 の場合も省略せず「文字1」と表す
- 入力は英小文字のみで構成される

## 入力

```
S
```

- 1 行目: 文字列 `S`（英小文字のみ）

## 出力

ランレングス圧縮した文字列を 1 行で出力してください。

## 制約

- `1 <= |S| <= 1000`
- `S` は英小文字 (a-z) のみで構成

## 入出力例

### 例 1

**入力**
```
aaabbc
```

**出力**
```
a3b2c1
```

**説明**
- `a` が 3 回連続 → `a3`
- `b` が 2 回連続 → `b2`
- `c` が 1 回連続 → `c1`

### 例 2

**入力**
```
a
```

**出力**
```
a1
```

**説明**
- `a` が 1 回だけなので `a1`

### 例 3

**入力**
```
abcde
```

**出力**
```
a1b1c1d1e1
```

**説明**
- すべて異なる文字なので各文字が 1 回ずつ

### 例 4

**入力**
```
aaaaaa
```

**出力**
```
a6
```

**説明**
- `a` が 6 回連続

### 例 5

**入力**
```
aabbccaabb
```

**出力**
```
a2b2c2a2b2
```

**説明**
- 途中で文字が変わると、そこで区切って新たにカウント
- `a` 2回、`b` 2回、`c` 2回、`a` 2回、`b` 2回

## ヒント

- 文字列を1文字ずつ走査し、前の文字と同じなら連続回数を増やす
- 文字が変わったら、それまでの文字と連続回数を出力に追加
- 最後の文字グループの処理を忘れずに
