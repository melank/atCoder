# ゼロの移動

**難易度**: ★★☆☆☆（初級）

**カテゴリ**: 配列操作（2ポインタ）

## 問題

整数配列 `nums` が与えられます。

配列内の **すべての 0 を末尾に移動** してください。ただし、**0 以外の要素の相対的な順序は保持** しなければなりません。

**重要**: 配列を**その場で（in-place）**修正してください。新しい配列を作成せずに解いてください。

## 入力

```
N
a_1 a_2 ... a_N
```

- 1 行目: 配列の要素数 `N`
- 2 行目: 配列の要素（スペース区切り）

## 出力

0 を末尾に移動した後の配列を、スペース区切りで 1 行で出力してください。

## 制約

- `1 <= N <= 10^4`
- `-10^9 <= a_i <= 10^9`

## 入出力例

### 例 1

**入力**
```
5
0 1 0 3 12
```

**出力**
```
1 3 12 0 0
```

**説明**
- 0 以外の要素 `1, 3, 12` の順序を保ったまま先頭に配置
- 0 が末尾に移動

### 例 2

**入力**
```
1
0
```

**出力**
```
0
```

**説明**
- 要素が 1 つだけで、それが 0 なのでそのまま

### 例 3

**入力**
```
3
1 2 3
```

**出力**
```
1 2 3
```

**説明**
- 0 がないので変化なし

### 例 4

**入力**
```
4
0 0 0 1
```

**出力**
```
1 0 0 0
```

**説明**
- 1 だけが先頭に移動し、0 が末尾に

### 例 5

**入力**
```
6
4 0 5 0 0 6
```

**出力**
```
4 5 6 0 0 0
```

**説明**
- `4, 5, 6` の順序を保って先頭に配置
- 3 つの 0 が末尾に移動

## ヒント

### 2ポインタアプローチ

1. `writePos` ポインタを用意（0 以外の要素を書き込む位置）
2. 配列を先頭から走査
3. 0 以外の要素を見つけたら、`writePos` の位置に書き込んで `writePos` を進める
4. 最後に、`writePos` 以降を 0 で埋める

### 別のアプローチ：スワップ

1. `left` ポインタ（次に 0 以外を置く位置）
2. `right` ポインタ（現在見ている位置）
3. 0 以外を見つけたら `left` と `right` の要素をスワップ

### 計算量

- 時間: O(N)
- 空間: O(1)（その場で修正するため）
